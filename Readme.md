# Task 2 - Templating + Bundler Implementation

##  Objective
Recreated Task 1 UI using templating engine and bundler.

---

##  Technologies Used
- Nunjucks (Templating Engine)
- Vite (Bundler)
- HTML, CSS, Bootstrap

---

##  Folder Structure
- src/templates → template files (.njk)
- src/assets → CSS
- dist → compiled HTML output

---

##  How to Run

npm install  
npm run build  

---

##  Output
dist/ folder contains:
- index.html  
- about.html  
- contact.html
